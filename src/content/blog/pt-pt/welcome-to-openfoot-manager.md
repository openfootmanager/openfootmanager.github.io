Temos o prazer de apresentar oficialmente o **Openfoot Manager**: um jogo gratuito e de código aberto de simulação de gestão de futebol.

## O que é o Openfoot Manager?

Openfoot Manager é um jogo para desktop onde assumes o papel de treinador de um clube de futebol. Tomarás decisões táticas, gerirás o teu plantel, cuidarás das finanças, contratarás funcionários, configurarás treinos e assistirás aos jogos a desenrolarem-se minuto a minuto: tudo alimentado por um sofisticado motor de jogo escrito em Rust.

Ao contrário das alternativas comerciais, o Openfoot Manager é:

- **100% gratuito e de código aberto**: licenciado sob GPLv3
- **Totalmente offline**: sem necessidade de internet, sem serviços na nuvem, sem DRM
- **Modificável**: ficheiros de dados JSON externos para ligas, equipas e jogadores personalizados
- **Leve**: aplicação desktop baseada em Tauri com binários pequenos

## A jornada até aqui

Este projeto começou em 2020 como um projeto hobby em Python, inspirado pelo [**Bygfoot**](https://bygfoot.sourceforge.io/new/), um clássico jogo de código aberto de gestão de futebol, e, claro, pelo [**Football Manager**](https://www.footballmanager.com/). Após várias iterações e uma reescrita completa em Rust, o jogo evoluiu para uma aplicação desktop moderna com uma interface baseada em React.

## O que vem a seguir

Estamos em desenvolvimento ativo, a trabalhar em direção ao primeiro lançamento alfa público. Os sistemas centrais estão operacionais (simulação de jogos, gestão de plantel, treino, staff, finanças, transferências e mais). Agora estamos focados em polir a experiência e expandir o conjunto de funcionalidades.

Fica atento a este blog para atualizações de desenvolvimento, notas de lançamento e novidades da comunidade.

## Participa

O Openfoot Manager é um projeto comunitário. Sejas programador, designer, tradutor ou fã de futebol, há um lugar para ti:

- **Dá uma estrela no repositório** no [GitHub](https://github.com/openfootmanager/openfootmanager)
- **Reporta bugs** ou **sugere funcionalidades** via [Issues](https://github.com/openfootmanager/openfootmanager/issues)
- **Contribui com código** — consulta o [Guia de Contribuição](https://github.com/openfootmanager/openfootmanager/blob/develop/CONTRIBUTING.md)

Vamos construir juntos o futuro da gestão de futebol de código aberto.
