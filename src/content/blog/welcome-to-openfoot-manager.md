We're excited to officially introduce **Openfoot Manager**: a free, open-source football management simulation game.

## What is Openfoot Manager?

Openfoot Manager is a desktop game where you step into the shoes of a football club manager. You'll make tactical decisions, manage your squad, handle finances, hire staff, configure training, and watch matches unfold minute-by-minute: all powered by a sophisticated match engine written in Rust.

Unlike commercial alternatives, Openfoot Manager is:

- **100% free & open source**: licensed under GPLv3
- **Fully offline**: no internet required, no cloud services, no DRM
- **Moddable**: external JSON data files for custom leagues, teams, and players
- **Lightweight**: Tauri-based desktop app with small binaries

## The journey so far

This project started back in 2020 as a Python hobby project, inspired by [**Bygfoot**](https://bygfoot.sourceforge.io/new/), a classic open-source football management game, and, of course, [**Football Manager**](https://www.footballmanager.com/). After several iterations and a complete rewrite in Rust, the game has evolved into a modern desktop application with a React-based UI.

## What's next

We're in active early development, working toward the first public alpha release. The core systems are operational (match simulation, squad management, training, staff, finances, transfers, and more). We're now focused on polishing the experience and expanding the feature set.

Stay tuned to this blog for development updates, release notes, and community news.
